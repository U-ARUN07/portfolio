## âš™ï¸ Usage (Run Locally)

If you want to run this project on your machine:

### 1. Clone the Repository
```bash
git clone https://github.com/U-ARUN07/portfolio.git
cd portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Configure Environment Variables
Create a `.env` file in the root directory and add your EmailJS keys (sign up at [emailjs.com](https://www.emailjs.com/)):
```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

### 4. Run Development Server
```bash
npm run dev
```
